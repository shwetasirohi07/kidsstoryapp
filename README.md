# Kids Stories App 📚✨

A beautiful, interactive Streamlit web app that generates fun and meaningful stories for children.

## Features
- **Custom Characters** — Enter any name for the main character
- **Age Groups** — Stories adapted for ages 3-5, 6-8, and 9-12
- **6 Themes** — Friendship, Magic, Animals, Adventure, Princess, Space
- **Story Lengths** — Short, Medium, or Long
- **Moral Values** — Each story ends with a meaningful lesson
- **Download** — Save the story as a text file

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Live App

Deployed on [Streamlit Community Cloud](https://streamlit.io/cloud).
